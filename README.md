# Deadlyze App

Desktop application for analyzing **DeadLock** game statistics using the DeadLock API.

This repository contains the source code for the application and scripts for interacting with the DeadLock API.  

Currently, the repository focuses on preparing Rust scripts to query the API and parse match data for analysis.

## Current Structure
```
deadlyze-app/
├── api-scripts/ # Rust scripts for querying DeadLock API and parsing game statistics
├── README.md # This file
└── LICENSE # CC BY-NC 4.0
```

## Purpose

- Learn and test API endpoints for DeadLock (e.g., player match history)
- Parse data to determine metrics such as win rate, best performance, etc.
- Prepare data for integration into the desktop application

## DeadLock API Reference

DeadLock API documentation: [https://api.deadlock-api.com/docs](https://api.deadlock-api.com/docs)
